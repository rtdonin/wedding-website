===============================
 Wedding Website Setup Guide
===============================

1. How to Replace Placeholder Images
------------------------------------
Each page currently has a placeholder image stored in the "images" folder.
Replace them with your chosen photos while keeping the same file names.

List of placeholders:
- landing.jpg   → Main hero image on the homepage
- faq.jpg       → Banner for FAQ page
- travel.jpg    → Banner for Travel Info page
- hotel.jpg     → Banner for Hotel Info page
- todo.jpg      → Banner for What To Do page
- rsvp.jpg      → Square RSVP image
- registry.jpg  → Banner for Registry page
- header.jpg    → Floral accent in the header
- footer.png    → Floral accent in the footer

2. Recommended Image Dimensions
-------------------------------
- Landing (landing.jpg): 1920 x 1080 px (hero background)
- FAQ (faq.jpg): 1200 x 600 px
- Travel (travel.jpg): 1200 x 800 px
- Hotel (hotel.jpg): 1200 x 800 px
- What To Do (todo.jpg): 1200 x 800 px
- RSVP (rsvp.jpg): 1000 x 1000 px (square)
- Registry (registry.jpg): 1200 x 800 px
- Header (header.jpg): 1920 x 300 px
- Footer (footer.png): 1920 x 200 px

3. File Types
-------------
- Use JPG for large photos (landing, travel, hotel, etc.)
- Use PNG for floral overlays (header/footer) to preserve transparency
- Optimize images under 500KB for faster load times

4. Customization Notes
----------------------
- Color scheme is emerald + pink (already styled in CSS).
- All common elements (navbar, footer) are generated dynamically with jQuery.
- You can replace text content directly in each HTML file.
- CSS automatically resizes and crops images responsively, so focus on choosing beautiful photos without worrying about alignment.
